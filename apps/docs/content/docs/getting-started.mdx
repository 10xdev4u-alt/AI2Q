# Getting Started with AIQL

AIQL (Artificial Intelligence Query Layer) is a high-performance intelligence driver for your database. It allows you to interact with your data using natural language, supporting multiple languages (Go, Python, C++) and databases (Postgres, MongoDB, Supabase).

## Installation

### CLI
Download the latest binary from our GitHub releases and add it to your path.

```bash
# Verify installation
aiql --help
```

### Go Driver
Install the Go driver in your project:

```bash
go get github.com/aiql/aiql-go
```

### Python Bindings
Install the Python package via pip:

```bash
pip install aiql
```

## Quick Start

1. **Crawl your schema**:
   ```bash
   aiql crawl --url "postgresql://user:pass@localhost:5432/mydb"
   ```

2. **Translate a query**:
   ```bash
   aiql translate --prompt "Get the top 5 users by signup date" --url "..."
   ```

3. **Run a migration**:
   ```bash
   aiql migrate --prompt "Add a bio column to the users table" --url "..."
   ```
